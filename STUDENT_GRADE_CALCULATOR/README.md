#STUDENT GRADE CALCULATOR
---------------------
Description:
-----
Calculates a student's percentage average,total mark and grade.
 
GAME EXECUTION:
------
For the game to execute run the following in the terminal (after cloning the CODSOFT repository):<br>
*java Main*
